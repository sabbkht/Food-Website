# food-website

A responsive restaurant/food ordering website with smooth scrolling, an interactive image slider, and a scroll-based navigation highlight. Built as a practice project to strengthen HTML, CSS, and JavaScript fundamentals.

## Features

* Fully responsive layout (desktop, tablet, mobile)
* Mobile-friendly toggle navbar
* Scroll-spy navigation (active link updates automatically while scrolling)
* Image slider for featured dishes (Swiper.js)
* Testimonial/review slider with multiple breakpoints
* Search popup form
* Page loader animation
* Clean, semantic HTML structure

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla)
* [Swiper.js](https://swiperjs.com/) for sliders
* Font Awesome for icons
